# nix-dev

nix development templates

## Usage

Initialize flake in new or existing project with

```sh
nix flake init --template github:frncsdrk/nix-dev#{TEMPLATE}
```

where `{TEMPLATE}` can be one of the following

- golang
- nodejs
- ziglang
